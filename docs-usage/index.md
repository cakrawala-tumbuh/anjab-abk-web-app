# ANJAB-ABK — Panduan Pengguna

Selamat datang di panduan pengguna aplikasi **ANJAB-ABK**.

Aplikasi ini membantu yayasan pendidikan menjalankan **Analisis Jabatan (ANJAB)** dan
**Analisis Beban Kerja (ABK)** untuk satu atau lebih satuan pendidikan. Aplikasi
mengelola data jabatan, panel ahli (SME), partisipan, dan **empat alat ukur** untuk
mengumpulkan data dari pegawai, lalu mengolahnya menjadi informasi jabatan dan beban kerja.

---

## Empat Alat Ukur

| Alat Ukur | Singkatan | Mengukur |
|---|---|---|
| **Task Inventory** | TI | Relevansi tugas per jabatan & rincian beban kerja (CalHR), melalui alur 3 tahap |
| **Time Study** | TS | Distribusi waktu aktivitas harian pegawai melalui log harian |
| **Demand-Control-Support** | DCS | Tuntutan, kendali, dan dukungan dalam pekerjaan (3 sub-skala, 42 item) |
| **Workplace Climate/Characteristics Profile** | WCP | Iklim & karakteristik tempat kerja (12 dimensi, 72 item) |

---

## Struktur Dokumentasi

Dokumentasi penggunaan dibagi menjadi dua jenis dokumen:

- **SOP (Standard Operating Procedure)** — prosedur baku *apa* yang harus dilakukan dan
  *urutannya*. Terbagi menjadi **SOP Persiapan** untuk masing-masing alat ukur dan
  **SOP Pelaksanaan Pengambilan Data**.
- **Instruksi Kerja (IK)** — langkah teknis *bagaimana* menggunakan aplikasi: detail
  klik tombol, mengisi formulir, dan menginput data.

!!! tip "Cara memakai dokumentasi ini"
    Gunakan **SOP** untuk memahami alur kerja secara keseluruhan, lalu buka **IK** terkait
    saat Anda perlu langkah teknis penginputan di aplikasi.
